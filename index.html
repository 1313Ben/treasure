    HTTP / AJAX

    //what is a HTTP(HyperText Transfer Protocol) -Request
It contains three to four parts:  
1. a verb or method (GET, POST, PATCH or DELETE)
2. a url (e.g: https://www.google.com/search?q=lewagon)
3. headers (metadata built by the client > User-Agent, Referer, Cookies, etc.)
4. a body (data coming from a form) for POST, PATCH and PUT requests

    // command-line tool to use

  // Do something once HTTP response is received
});
    // if you want to do something with response data
fetch(url)
  .then(response => response.json())
  .then((data) => {
  console.log(data); // `data` is now a JS object
});

    // example for a POST
The HTTP method (aka verb)
The HTTP headers (telling the server we’re sending some JSON)
The HTTP body, with the actual JSON being posted
fetch(url, {
  method: "POST",
  headers: {"Content-Type": "application/json" },
  body: JSON.stringify(a_js_object)
})

    //How do you toggle an .img-circle class on all img on click?
1. Select all images with querySelectorAll
2. Then bind a listener to the click event
3. The callback for this listener uses event.
currentTarget and updates its classes. 
document.querySelectorAll("img").forEach((img) => {
  img.addEventListener("click", (event) => {
  event.currentTarget.classList.toggle("img-circle");
  });
});
