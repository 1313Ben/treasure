    HTTP / AJAX

    //what is a HTTP(HyperText Transfer Protocol) -Request
It contains three to four parts:  
1. a verb or method (GET, POST, PATCH or DELETE)
2. a url (e.g: https://www.google.com/search?q=lewagon)
3. headers (metadata built by the client > User-Agent, Referer, Cookies, etc.)
4. a body (data coming from a form) for POST, PATCH and PUT requests
